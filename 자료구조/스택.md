## 스택 코드 구현 (배열 사용)

```java
class Stack {
    int[] arr = new int[10001];
    
    int cnt = 0;
    
    void push(int value) {
        arr[cnt++] = value;
    }
    
    int pop() {
        if(cnt == 0) return -1;
        else return arr[--cnt];
    }
    
    int size() {
        return cnt;
    }
    
    int empty() {
        return cnt == 0 ? 1 : 0;
    }
    
    int top() {
        if(cnt == 0) return -1;
        else return arr[cnt-1];
    }
}
```
